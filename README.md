# IDT-calculator

ローイング（ボート）競技における重要なトレーニング指標である **IDT（Ideal Time: 目標艇速比）** を、簡便に算出するためのWeb電卓ツールです。

名古屋大学が提唱する計算式をベースに、2000m TT（タイムトライアル）の記録だけでなく、日頃のトレーニングで行われることの多い **20分TT** の結果からもIDTを推定・算出できるオリジナルの調整を加えています。

---

## 🌟 主な機能と特徴

- **2つの入力モードに対応**
  - **2000m TT**: 2000mの完漕タイム、体重、性別から直接IDTを算出します。
  - **20分 TT**: 20分エルゴでの「平均500mペース」と体重、性別からIDTを推定します。
- **複数データの一括計算・管理**
  - 「＋ 行を追加」ボタンで入力行を自由に追加し、複数クルーや複数回の測定結果を一覧で一括計算できます。
  - 不要な行は「✕」ボタンで個別に削除でき、全体のクリアもワンタップで行えます。
- **モダンでレスポンシブなUI/UX**
  - スマートフォン（iOS / Android）のブラウザでのエルゴサイドでの利用を想定し、押しやすいボタン幅や入力ズームを防ぐフォントサイズ設計にしています。
  - システム設定や好みに応じてワンタップで切り替えられる **ダークモード / ライトモード** を搭載。
  - `Enter` キーでのクイック計算に対応しています。

---

## 📐 計算ロジックについて

### 1. IDT (Ideal Time) 計算式
本ツールでは、[名古屋大学の計算式](https://marine.isee.nagoya-u.ac.jp/labhp/member/aiki/idte.html)を採用しています。体重によるエルゴパフォーマンスの影響を補正し、公平な艇速適性を評価します。

### 2. 20分TTからの推定ロジック
20分間の平均500mペースから2000mTT相当の秒数を逆算するため、以下の補正係数を導入しています。
- **2000m相当換算秒数** = `20分TT平均500mペース(秒) * 4 * 補正係数`
  - **男性補正係数**: `0.94`
  - **女性補正係数**: `0.96`
  ※男女一律の補正ではなく、性別ごとに最適化した係数を用いることで、女性の推定IDTが不自然に高くなる問題を軽減しています。

## 🚀 使い方

単一の HTML ファイル (`index.html`) のみで動作するため、特別なサーバーやビルド環境は不要です。

1. 本リポジトリをクローンまたはダウンロードします。
2. `index.html` を任意のウェブブラウザで開きます。
3. タイム（例: `7:42.6` や `2:05.3`）、体重を入力し、性別を選択して「計算」ボタンを押します。
